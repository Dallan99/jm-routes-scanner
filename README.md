# 🚚 JM Routes Scanner

Sistema de recebimento, conferência e triagem de rotas desenvolvido para a **JM Transportes**, com foco em operações de Last Mile e e-commerce.

O objetivo do projeto é substituir controles manuais em Excel por uma plataforma moderna, segura e escalável para gestão operacional.

---

# 📋 Sobre o Projeto

O JM Routes Scanner foi desenvolvido para otimizar o processo de recebimento de rotas, conferência de cargas e acompanhamento operacional em tempo real.

O sistema permite maior rastreabilidade, redução de erros operacionais e aumento da produtividade das equipes de triagem.
